# Rizz Assist

AI-powered dating wingman that analyzes her psychology and adapts responses to her communication style.

## Features

- **Psychological profiling** — input details about her and the AI builds a structured personality profile with trait scores (playfulness, flirtiness, directness, etc.)
- **Adaptive suggestions** — reply suggestions are tuned to her communication style, attachment type, and personality
- **Message rephrasing** — type what you want to say, get 3 polished versions matched to her vibe
- **Red flag detection** — analyze her messages or your drafts for manipulation, desperation, boundary issues
- **Smart memory** — key facts (pet names, interests, plans) are stored permanently; old conversations auto-compress to save tokens
- **Token-optimized** — ~500-700 tokens per API call regardless of conversation length

## Setup

1. Open the app at [your GitHub Pages URL]
2. Enter your API key (OpenAI or DeepSeek)
3. Fill in details about her — the AI builds a psychological profile
4. Start chatting

## How it works

- **Her message → Suggest** — get 3 reply options matched to her style
- **Type your draft → Suggest** — get 3 rephrased versions of what you want to say
- **Red flags** — analyze any message for manipulation patterns or healthy signals
- **Rewrite** — polish your draft message

## Privacy

- Your API key stays in your browser's localStorage — never sent to any server except the AI provider you choose
- All data is stored locally in your browser
- No analytics, no tracking, no backend

## Supported providers

- OpenAI (GPT-4o-mini, GPT-4o, etc.)
- DeepSeek
- Any OpenAI-compatible API via custom URL
